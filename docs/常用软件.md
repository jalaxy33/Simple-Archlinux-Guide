# 常用软件

## 【浏览器】

按自己的喜好选择:

### firefox

需要同时安装中文语言包

```sh
sudo pacman -S --needed firefox firefox-i18n-zh-cn
```

### Chrome

AUR安装：

```sh
yay -S --needed google-chrome
```

### Zen

需要同时安装中文语言包

```sh
yay -S --needed zen-browser zen-browser-i18n-zh-cn
```

### Brave

```sh
sudo pacman -S --needed brave-bin
```

## 【软件管理器】

### pacman管理

```sh
sudo pacman -S --needed pacman-contrib pamac-aur
```

`pacman-contrib` 提供pacman的一些额外功能，比如 checkupdates 用来检查更新

`pamac-aur` pacman的GUI界面

### AUR管理

```sh
yay -S --needed pacseek
```

`pacseek` 提供TUI（终端用户交互工具）用来可视化搜索官方源和aur软件包的，安装需要网络环境

### flatpak管理

```sh
flatpak install io.github.flattool.Warehouse com.github.tchx84.Flatseal io.github.fabrialberio.pinapp
```

`Warehouse` 用来管理flatpak的源、软件、属性、用户数据之类的

`Flatseal` 管理flatpak应用的权限、环境变量之类的

`pinapp` 用来管理软件快捷方式（`.desktop`），可以用来设置软件环境变量、隐藏不要的快捷方式等

## 【office办公套件】

有以下替代选择：

### onlyoffice

国产的开源办公软件，比较轻量且无广告。

```sh
sudo pacman -S --needed onlyoffice-bin
```

### wps

需要额外安装中文语言包

```bash
yay -S --needed wps-office-cn wps-office-mui-zh-cn
```

### libreoffice

知名开源办公软件，有`fresh`尝鲜版和`still`稳定版两个版本，需要安装对应的中文语言包。

- 尝鲜版

  ```sh
  sudo pacman -S --needed libreoffice-fresh libreoffice-fresh-zh-cn
  ```

- 稳定版

  ```sh
  sudo pacman -S --needed libreoffice-still libreoffice-still-zh-cn
  ```

## 【音视频 & 影像处理】

- 官方源

  ```sh
  sudo pacman -S --needed easyeffects curtail switcheroo handbrake
  ```

  `easyeffects` 设置麦克风降噪、混响、音响音效之类的

  `curtail` 压缩图片大小

  `switcheroo` 变更图片格式

  `handbrake` 视频转码

- flatpak软件

  ```sh
  flatpak install io.gitlab.theevilskeleton.Upscaler com.dec05eba.gpu_screen_recorder io.gitlab.adhami3310.Footage
  ```

  `upscaler` 图片超分

  `gpu_screen_recorder` 类似nvidiaApp的录屏软件

  `footage` 简单的视频剪辑、压制软件

## 【网络工具】

- 官方源

  ```sh
  sudo pacman -S --needed localsend
  ```

  `localesend` 局域网传输文件

- flatpak软件

  ```sh
  flatpak install com.github.unrud.VideoDownloader net.agalwood.Motrix
  ```

  `video downloader`下载youtube/bilibili 144p～8k视频

  `motrix` 跨平台下载器，支持磁力链接和 aria2 加速

## 【娱乐】

- 官方源

  ```sh
  sudo pacman -S --needed komikku shortwave
  ```

  `komikku` 看漫画

  `shortwave` 收听电台

- AUR

  ```sh
  yay -S --needed splayer
  ```

  `splayer` 网易云音乐的第三方客户端，安装需要网络环境

- flatpak软件：

  ```sh
  flatpak install io.github.Predidit.Kazumi com.spotify.Client
  ```

  `kazumi` 追番

  `spotify` 听歌软件
