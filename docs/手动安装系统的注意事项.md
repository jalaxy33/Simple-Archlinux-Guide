# 手动安装系统的注意事项

## 推荐：创建不备份的 btrfs 子卷

在[创建btrfs子卷](https://github.com/SHORiN-KiWATA/Shorin-ArchLinux-Guide/wiki/安装ArchLinux#创建btrfs子卷)的步骤，推荐创建一个不备份的子卷，专门用来存放一些不需要快照备份的文件，比如游戏文件、代码、数据集等。这样可以避免备份的时候占用大量空间，同时也方便管理。

我的习惯是创建一个叫 `@Unsnap` 的子卷，挂载到 `/mnt/Unsnap` 下。

```sh
# 创建子卷
btrfs subvolume create /mnt/@Unsnap

# 经过若干步骤...

# 挂载
mount --mkdir -t btrfs -o subvol=/@Unsnap,compress=zstd /dev/nvme0n1p2 /mnt/Unsnap
```

## 多个硬盘怎么分区

关键是先在硬盘分区上创建对应的 btrfs 子卷，其他步骤和单硬盘安装一样。

假设有两个硬盘，分别是 `/dev/nvme0n1` 固态和 `/dev/sda` 机械盘，准备将系统安装在固态上。

1. 格式化分区。参考教程里[创建btrfs子卷](https://github.com/SHORiN-KiWATA/Shorin-ArchLinux-Guide/wiki/安装ArchLinux#硬盘分区)之前的步骤，固态系统盘的步骤与教程相同，主要看机械盘的步骤：
   - 创建硬盘分区，类型选择 Linux filesystem
     ```sh
     cfdisk /dev/sda
     ```
   - 格式化为 btrfs 文件系统
     ```sh
     mkfs.btrfs -f /dev/sda1  # 分区号可能不是 sda1，根据实际情况修改
     ```

2. **关键步骤**：创建各个分区的 btrfs 子卷。参考[教程](https://github.com/SHORiN-KiWATA/Shorin-ArchLinux-Guide/wiki/安装ArchLinux#创建btrfs子卷)，固态系统盘与教程里的步骤一样，这里主要说一下机械盘的步骤：
   - 将机械盘挂载到 `/mnt` 目录下
     ```sh
     # 将机械盘挂载到 /mnt 目录下
     mount /dev/sda1 /mnt
     ```
   - 创建 btrfs 子卷，起一个自定义的名字，后续挂载的时候会用到
     ```sh
     btrfs subvolume create /mnt/@SDA
     ```
   - 可选：确认
     ```sh
     btrfs subvolume list /mnt
     ```
   - 取消挂载
     ```sh
     umount /mnt
     ```

3. 之后的[正式挂载](https://github.com/SHORiN-KiWATA/Shorin-ArchLinux-Guide/wiki/安装ArchLinux#正式挂载)的步骤与教程里的一样，注意在挂载的时候，机械盘的挂载路径要改成之前创建的子卷路径：

   ```sh
   # 挂载机械盘的子卷到 /mnt/SDA 目录下
   mount --mkdir -t btrfs -o subvol=/@SDA,compress=zstd /dev/sda1 /mnt/SDA
   ```

4. 可选：复查挂载情况

   ```sh
   df -h
   ```

## 选择linux内核

在[安装系统](https://github.com/SHORiN-KiWATA/Shorin-ArchLinux-Guide/wiki/安装ArchLinux#安装系统)这一步，可以将 `linux` 替换成其他内核，如 `linux-zen`、`linux-lts` 等。

这一步可以把对应的内核头文件也一起装上：

```sh
# linux内核
pacstrap -K /mnt linux linux-headers

# linux-zen内核
pacstrap -K /mnt linux-zen linux-zen-headers

# linux-lts内核
pacstrap -K /mnt linux-lts linux-lts-headers
```

❗需要注意的是 `linux` 和 `linux-zen` 这两个内核会相互冲突，不能同时安装。

## 用 gvim 替换 vim 编辑器

`gvim`提供了更多的功能，包含了`vim`的全部功能，两者不能同时安装。在[安装系统](https://github.com/SHORiN-KiWATA/Shorin-ArchLinux-Guide/wiki/安装ArchLinux#安装系统)这一步，可以将 `vim` 替换成 `gvim`：

```sh
pacstrap /mnt gvim
```

安装后，`vim`命令会自动链接到 `gvim`，所以可以直接使用 `vim` 命令来编辑文件。
