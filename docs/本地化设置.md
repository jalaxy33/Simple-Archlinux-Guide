# 本地化设置

## 1. 生成本地化设置

1.  编辑 `locale.gen` 配置文件

    ```sh
    sudo vim /etc/locale.gen
    ```

    `/` 左斜杠键进行搜索；`i` 键进入编辑模式；取消`en_US.UTF-8 UTF-8`和`zh_CN.UTF-8 UTF-8`的注释；`esc`退出编辑模式；`:wq`保存并退出。

2.  生成本地化设置

    ```sh
    sudo locale-gen
    ```

3.  设置默认系统语言为英文

    编辑 `/etc/locale.conf` 文件，设置 `LANG` 变量为 `en_US.UTF-8`：

    ```sh
    sudo vim /etc/locale.conf
    ```

    添加以下内容：

    ```
    LANG=en_US.UTF-8
    ```

    保存并退出。

    > 后面可能还会改，先设置一个绝对不会出问题的默认值。

## 2. 设置系统语言

有两种选择，两种都可以：

1. 设置系统语言为中文
2. 设置系统语言为英文，单独为每个应用设置中文

📝 下面是所有桌面环境都适用的方法，Niri、Hyprland等窗口管理器也可以直接在配置文件里设置环境变量。

### 选择一：设置系统语言为中文

- 打开 `/etc/locale.conf` 文件：

  ```sh
  sudo vim /etc/locale.conf
  ```

  也可以编辑用户空间的 `~/.config/locale.conf` 文件，这样只会影响当前用户：

  ```sh
  vim ~/.config/locale.conf
  ```

- 设置语言相关的环境变量，两种方式：

  ```sh
  LANG=en_US.UTF-8
  LC_MESSAGES=zh_CN.UTF-8
  ```

  或者

  ```sh
  LANG=zh_CN.UTF-8
  LC_CTYPE=en_US.UTF-8
  ```

### 选择二：系统语言为英文，单独为每个应用设置中文

关键🔑就是**设置环境变量**，将 `LC_MESSAGES` 设置为 `zh_CN.UTF-8`。

#### 如果应用以快捷方式打开（.desktop）

关键：编辑 `.desktop` 文件修改 `Exec=` 后面的命令。

1. 找到应用的 `.desktop` 文件，通常位于 `/usr/share/applications/` 或 `~/.local/share/applications/` 目录下。

   ⚠️ 注意：不要直接修改 `/usr/share/applications/` 里的文件，最好复制一份到 `~/.local/share/applications/` 目录下进行修改。用户目录下的 `.desktop` 文件优先级更高，修改后不会影响系统其他用户。

2. 用文本编辑器打开 `.desktop` 文件，找到 `Exec=` 字段，在命令前添加环境变量设置：

   ```sh
   Exec=env LC_MESSAGES="zh_CN.UTF-8" your_application_command
   ```

之后启动这个应用时，它就会使用中文界面了，有些应用可能需要注销或重启才能生效。其他环境变量也可以用同样的方式设置。

#### 如果应用以命令行启动

在命令前添加环境变量即可：

```sh
LC_MESSAGES="zh_CN.UTF-8" <命令>
```

为方便也可以添加别名：

```sh
# 以vscode为例
alias code='LC_MESSAGES="zh_CN.UTF-8" code'
```

除了用 `alias`，也可以定义函数，fish shell 还可以用 `abbr` 来设置命令自动替换。
